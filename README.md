# griot
A tokenization library for NLP tasks!
## Description


## Installation

```bash
pip install git+https://github.com/vertigoaway/griot.git
```
OR
```bash 
git clone https://github.com/vertigoaway/griot.git
cd griot
pip install .
```

## Features
- Lightweight
- No dependencies
- Manages vocabulary dictionaries for you
- Simplistic and easy to understand functions
- Spaced word support
- MyPy compatible
- Character level support
- Positional encoding and decoding tools (soon)
- One-hot encoding and decoding tools (soon)
